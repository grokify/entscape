## [v0.1.0] - 2026-03-09

### Highlights

- Interactive entity-relationship diagram visualization for Ent.go schemas
- Generate standalone HTML files deployable to GitHub Pages
- NPM module with Cytoscape.js for custom integrations

- JSON schema types for entscape format with Go types and JSON Schema generation ([`9d80af5`](https://github.com/grokify/entscape/commit/9d80af5))

- Public `AddSourceLinks` method for HTML generator ([`1d2ca4d`](https://github.com/grokify/entscape/commit/1d2ca4d))
